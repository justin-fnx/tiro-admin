## 1. 디자인 시스템

### 1.1 컬러 팔레트

```css
/* 기본 색상 (shadcn/ui 기반) */
--background: 0 0% 100%;           /* 배경 (흰색) */
--foreground: 240 10% 3.9%;        /* 텍스트 (검정) */

--primary: 240 5.9% 10%;           /* 기본 액션 색상 */
--primary-foreground: 0 0% 98%;

--secondary: 240 4.8% 95.9%;       /* 보조 색상 */
--secondary-foreground: 240 5.9% 10%;

--muted: 240 4.8% 95.9%;           /* 비활성 영역 */
--muted-foreground: 240 3.8% 46.1%;

--accent: 240 4.8% 95.9%;          /* 강조 색상 */
--accent-foreground: 240 5.9% 10%;

--destructive: 0 84.2% 60.2%;      /* 위험/삭제 (빨강) */
--destructive-foreground: 0 0% 98%;

--border: 240 5.9% 90%;            /* 테두리 */
--input: 240 5.9% 90%;             /* 입력 필드 */
--ring: 240 5.9% 10%;              /* 포커스 링 */
```

### 1.2 상태별 색상

| 상태 | 배경색 | 텍스트 | 용도 |
|------|--------|--------|------|
| Success | `bg-green-100` | `text-green-800` | 완료, 활성화 |
| Warning | `bg-yellow-100` | `text-yellow-800` | 경고, 대기 |
| Error | `bg-red-100` | `text-red-800` | 실패, 오류 |
| Info | `bg-blue-100` | `text-blue-800` | 정보, 진행중 |
| Neutral | `bg-gray-100` | `text-gray-800` | 기본, 비활성 |

### 1.3 타이포그래피

| 요소 | 클래스 | 용도 |
|------|--------|------|
| 페이지 제목 | `text-3xl font-bold` | 페이지 최상단 제목 |
| 섹션 제목 | `text-xl font-semibold` | 카드/섹션 제목 |
| 서브 제목 | `text-lg font-medium` | 서브섹션 제목 |
| 본문 | `text-sm` | 일반 텍스트 |
| 캡션 | `text-xs text-muted-foreground` | 설명, 부가 정보 |

### 1.4 간격 시스템

| 크기 | 값 | 용도 |
|------|-----|------|
| xs | 4px (p-1) | 인라인 요소 간격 |
| sm | 8px (p-2) | 밀집된 요소 |
| md | 16px (p-4) | 기본 간격 |
| lg | 24px (p-6) | 섹션 간격 |
| xl | 32px (p-8) | 큰 섹션 분리 |

### 5.1 통계 카드 (StatCard)

```typescript
interface StatCardProps {
  title: string;
  value: string | number;
  change?: {
    value: number;
    type: 'increase' | 'decrease';
  };
  icon?: LucideIcon;
}
```

### 5.2 데이터 테이블 (DataTable)

```typescript
interface DataTableProps<T> {
  columns: Column<T>[];
  data: T[];
  pagination?: {
    page: number;
    limit: number;
    total: number;
  };
  onPageChange?: (page: number) => void;
  onRowClick?: (row: T) => void;
  loading?: boolean;
}
```

- 사이드바: 전체 표시

- 테이블: 전체 컬럼 표시
